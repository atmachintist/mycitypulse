## 🔍 Official Data Sources

### Primary Sources (Use These)

1. **State Election Commission Official Website**
   - URL: https://sec.gujarat.gov.in/
   - Contains all official notifications and publications

2. **Poll Monitoring System (Most Detailed)**
   - URL: https://sec-poll.gujarat.gov.in/Index.aspx
   - Instructions: 
     - Select Election Type: Municipal Corporation
     - Select Year: 2026
     - Select City/Corporation
     - Click "હરિફ ઉમેદવાર" (Candidates) to see all candidates by ward/party

3. **Municipal Corporation Elections Page**
   - URL: https://sec.gujarat.gov.in/mun-cor-gen-ele.htm
   - Contains ward-wise reservations, delimitations, and official announcements

## 📝 How to Get Verified Candidate Data

### Step 1: Access SEC Poll Monitoring System
1. Visit: https://sec-poll.gujarat.gov.in/Index.aspx
2. Select "Municipal Corporation" from dropdown
3. Select "2026" as the year
4. Choose your desired city/corporation

### Step 2: View Candidate Details
- Click "હરિફ ઉમેદવાર" (Candidates) column
- This shows ALL candidates across all parties
- Data includes:
  - Candidate name
  - Party affiliation
  - Ward number
  - Nomination status

### Step 3: Download Data
- Most systems allow export to Excel/CSV
- Verify data against official sources
- Note: Updates may be ongoing until nomination deadline

## 🔄 Website Update Workflow

### For Each Municipal Corporation:

1. **Access SEC Poll System**
   - Navigate to corporation page
   - Download candidate list (if export available)
   - Screenshot ward-wise reservations page

2. **Cross-Verify with Party Sources**
   - Check BJP official announcements
   - Check Congress/AAP/other party announcements
   - Note any discrepancies

3. **Update Website**
   - Add total candidate count per party
   - Update verified candidate list
   - Link to official SEC page for detailed ward-wise info
   - Mark verification date and source

4. **Add to Spreadsheet**
   - Use provided Excel template
   - Fill in: Corporation name, total wards, candidates by party, source
   - Include direct link to SEC Poll Monitoring System for that city
